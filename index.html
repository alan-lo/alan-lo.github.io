Rails Blog

The project is to implement a personal blog with ruby on rails. It features the ability for users to add, view , edit and delete posts and comments. The blog has complete user authentication and authorization. User is able to sign in with his/her account and submit and edit their own post. They can also like or dislike a post.

CodeCore Drillz

The project is to implement learning app. It features fully implemented user account and password reset system. Sign-in users have the ability to create drills and drills groups. Also, they can run and create drill groups. If the users answer all the correct drills within in a set, they will get a badge like donut. There is a leaderboard page to show all the top point collectors. The users can attempt a particular drill unlimited number of times. Points rewarded depend of the difficulty level of each drill group.

Role: Back-end development for the points and reward system

Technologies used: Ruby, Rails framework, JavaScript, Bootstrap, HTML3, CSS3
